PROJECT 2

TEXT EDITOR

The application is a sophisticated text editor developed using the .NET framework on Microsoft Visual Studio, leveraging the power of C++. As the user begins typing, the editor promptly generates suggestions, enhancing the overall writing experience. Additionally, it incorporates an efficient file-saving mechanism, utilizing Huffman encoding to store the text in an encoded format. Upon loading a text file, the editor seamlessly decodes the content and presents it to the user in its original form.
